With computer coding, I feel like Houdini. My stage is my laptop monitor as I active Browser Sync or Visual Studio Code. Whether it is a printf( "Hello, world!" ); from my first computer science course or a complicated 400-line code to create an advance spell-checker at minimal runtime in C, every line of code has a tangible effect. Although many find coding boring, I find it as fascinating as every refresh or run command I use to see my code growing before my eyes.

I started programming in early 2018 (or what I call programming at least). Before that, I have been an amused wanderer looking behind glass or just a tip-toe dancer sinking the tips of my toes in before rushing them out quickly (and I did that a lot). My first programming language was C++ from an online blog, followed by basics of C online (I know, weird timeline). However, my first actual experience, that of 2018, was with Javascript on Code Academy and Khan Academy, followed by Swift through CodeWithChris YouTube channel. Watching Swift content exposed me to the ideas of model-view-control patters, libraries, frameworks, UI design, handling events, and creativity of design.

Now, I work with HTML, CSS, Javascript, Python, C, C++, MySql, PHP, and more. So, welcome friend to the stage of The Great Houdini.

Linked Lists Library in C

A linked list is an array-like data structure that does not need a pre-defined size (i.e. can be expanded) since its memory is dynamically allocated. However, unlike an array, a linked list does not have random memory access. This library brings linked lists to C and contains functions to declare a new linked list, append items at certain indexes, fetch elements, print a list, and more.

jQuery Gallery Generator

A gallery designed in HTML, CSS, and Javascript (jQuery), where each gallery item expands upon hovering. Position and size of the hover item is calculated entirely in jQuery. To include this gallery in your website, a lot of preprocessing steps are needed that can be handled using the gallery generator, which is written in Python. The generator takes the images, parses them and matches their names accordingly using PILLOW, and then generates the final HTML with the help of REGEX.

College Lookup

Credit Card Validator

Credit cards use a validation algorithm that utilizes the last digit of the credit card number. This algortithm is called the Luhn Algortithm and could be used to verify credit card validity. The program, written in C, takes in a credit card number, validates it, and checks if it is one of 11 different credit card companies. This projects is part of CS50 assignments.
